## Project Description
A comprehensive JavaScript practice repository covering core language fundamentals, data persistence with localStorage, and professional code quality practices. Features two standalone mini-projects: an interactive Calculator and a Student Grade Tracker, plus a refactored To-Do List with persistent storage.

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- localStorage / sessionStorage
- ESLint & Prettier (code quality)

## Features
### Core JavaScript Exercises
- ✅ Variables (`let`, `const`, `typeof`)
- ✅ Data types & operators (strings, numbers, comparisons)
- ✅ Functions (declarations, arrows, defaults, parameters)
- ✅ Control flow (if/else, switch, loops, FizzBuzz)
- ✅ Arrays & Objects (methods: `map`, `filter`, `reduce`, `find`)
- ✅ Array of objects & real-data patterns

### Mini-Projects
#### 🧮 Interactive Calculator
- Basic operations: `+`, `-`, `*`, `/`, `%`, `**`
- Division-by-zero handling
- Keyboard support & responsive UI
- Clean, accessible design

#### 📊 Student Grade Tracker
- Add students with multiple subject grades
- Calculate individual & class averages
- Letter grade conversion (A-F)
- Identify top performers & struggling students
- Generate formatted report cards
- Data persists with localStorage

### Refactored To-Do List (Final Deliverable)
- ✅ Todos persist across page refreshes
- ✅ Completed state & filter preference saved
- ✅ Modular architecture: `state.js`, `storage.js`, `ui.js`, `utils.js`
- ✅ Immutable state updates
- ✅ ESLint/Prettier configured for code quality

### Code Quality Practices
- Meaningful variable/function names
- Single-responsibility functions
- No magic numbers (named constants)
- Proper error handling
- Consistent formatting with Prettier
